## 模块一：研究背景与问题定义

### 动机
现代 T2I（文本到图像）模型现在能够渲染清晰的段落级文本，这开辟了一类全新的滥用途径。攻击者可以强迫 T2I 系统生成**包含有害文本载荷**的图像（如欺诈性文件），并将其嵌入视觉上无害的场景中。这与传统的"描绘式越狱"（生成视觉上不良的图像）有本质区别。

### 科学问题
**T2I 模型的文本渲染能力是否引入了新的安全攻击面？能否构造一种攻击，使 T2I 系统生成视觉无害但包含有害文本内容的图像，同时绕过多阶段安全过滤器？**

### 直觉洞见
- T2I 模型的文本渲染能力越来越强，但安全对齐（safety alignment）主要关注**视觉内容**，忽略了**文本内容**
- "铭文式越狱"是一个被忽视的攻击类别：图像看起来完全正常，但其中渲染的文字包含有害信息
- 传统越狱技术面向粗粒度的视觉操作，在保持字符级保真度的同时绕过安全过滤器是一个新挑战
- 攻击可以分解为三个正交层，将复杂的联合优化简化为可处理的子问题

## 模块二：核心方法

### 技术贡献
提出 **Etch**（铭刻）攻击框架，首次系统化"铭文式越狱"攻击：

1. **攻击分类学**：区分"描绘式越狱"（depictive jailbreak）和"铭文式越狱"（inscriptive jailbreak）
2. **三层对抗提示分解**：
   - **语义伪装层（Semantic Camouflage）**：让整体提示看起来无害
   - **视觉空间锚定层（Visual-Spatial Anchoring）**：指定有害文本在图像中的位置和布局
   - **排版编码层（Typographic Encoding）**：确保有害文本被准确渲染
3. **VLM 批评循环**：视觉语言模型评估生成图像，将失败定位到具体层，提出针对性修订

### 方法框架
```
有害文本载荷（如欺诈文件内容）
    ↓
Etch 三层分解：
├── 语义伪装层 → "一个古典风格的信函"（无害外观）
├── 视觉锚定层 → "文本居中，A4纸背景"
└── 排版编码层 → "精确渲染以下文字: ..."
    ↓
零阶优化循环：
├── T2I模型生成图像
├── VLM批评：检查三层目标达成度
├── 定位失败层 → 针对性修订
└── 迭代至成功
    ↓
最终输出：视觉无害但含有害文本的图像
```

## 模块三：技术细节

### 三层分解的数学基础

**核心思想**：将对抗提示空间 P 分解为三个功能正交的子空间：
- P = P_semantic × P_spatial × P_typographic
- 联合优化 max f(p) 简化为三个子问题的迭代优化

**零阶优化循环**：
1. 初始化三层提示组件
2. T2I 模型生成图像
3. VLM 批评模型评估图像：
   - 语义伪装是否通过？（图像整体是否看起来无害）
   - 空间锚定是否准确？（文本位置是否正确）
   - 排版编码是否保真？（渲染的文字是否准确）
4. 将失败定位到具体层
5. 只修改失败层的提示组件
6. 重复直到攻击成功

### 关键特征
- **黑盒攻击**：不需要访问 T2I 模型的内部参数
- **功能正交分解**：每层独立优化，避免高维联合搜索
- **VLM 驱动的迭代**：利用 VLM 的理解能力精确定位失败原因

### 与传统越狱的区别

| 属性 | 描绘式越狱 | 铭文式越狱（Etch） |
|------|------------|-------------------|
| 攻击目标 | 视觉内容（有害图像） | 文本内容（有害文字） |
| 图像外观 | 明显不当 | 视觉上无害 |
| 利用能力 | T2I 图像生成 | T2I 文本渲染 |
| 检测难度 | OCR + 视觉检测 | 需要 OCR + 语义分析 |
| 实际危害 | 冒犯性图像 | 欺诈文件、虚假证书等 |

## 模块四：实验设置与结果

### 实验设置
- **评估模型**：7 个 T2I 模型
- **评估基准**：2 个基准
- **指标**：攻击成功率（ASR）

### 核心结果

| 指标 | 结果 |
|------|------|
| 平均攻击成功率 | **65.57%** |
| 峰值攻击成功率 | **91.00%** |
| 相对基线提升 | 显著优于现有方法 |

### 跨模型对比
- Etch 在所有 7 个模型上均显著优于现有越狱基准
- 攻击效果在不同模型架构间表现一致

## 模块五：局限性分析

1. **文本渲染能力依赖**：攻击效果依赖 T2I 模型的文本渲染质量，对文本渲染能力较弱的模型效果可能下降
2. **字符级保真度挑战**：长文本或特殊字符的准确渲染仍是挑战
3. **VLM 批评模型的限制**：VLM 的评估准确性直接影响优化循环的效率
4. **防御空白**：论文揭示了攻击面但未提出完整的防御方案
5. **实际滥用场景**：欺诈文件等场景的实际可行性需要考虑后续分发链条

## 📌 核心启示

**对 AI 安全实践的影响**：
- **新攻击类别**：铭文式越狱是一个被忽视但具有高实际危害的攻击类别
- **文本渲染 = 新攻击面**：T2I 模型的文本渲染能力进步带来的安全风险需要重视
- **排版感知防御**：现有 T2I 安全过滤器需要引入 OCR + 语义分析来检测铭文式攻击
- **多模态防御需求**：需要同时分析图像的视觉内容和渲染的文本内容
- **实际威胁**：欺诈性文件生成可能是比冒犯性图像更具现实危害的威胁
