## 一、背景与问题

### 动机 (Motivation)
文生图（T2I）生成模型广泛部署在各种创意工具和在线平台上，为防止滥用，这些系统严重依赖安全过滤器和内容审核管道。然而，现有的研究多集中于复杂的对抗优化，忽视了最朴素的用户输入层面的威胁。

### 核心科学问题 (Core Scientific Question)
现代文本到图像生成模型的安全过滤器是否能抵御仅需自然语言修改（无需模型访问、对抗训练或优化）的“低成本”越狱攻击？

### 直觉洞见 (Intuitive Insight)
越狱 T2I 模型不一定需要复杂的梯度计算。就像人们用“艺术探讨”作为借口来讨论敏感话题一样，攻击者只需用自然语言给违规概念穿上“伪装外衣”（如伪装成教育材料、艺术重构、生活方式美学等），就能骗过只看字面意思的审核系统，同时模型依然能“听懂”并生成违规图像。

---

## 二、核心方法

（基于摘要推断）

**视觉越狱分类学（Taxonomy of Visual Jailbreak Techniques）**
提出了一套基于语义伪装的低成本自然语言越狱策略分类：
1. **Artistic Reframing (艺术重构)**: 将敏感内容包装成特定艺术流派或油画描述。
2. **Material Substitution (材质替换)**: 使用隐喻材质（如大理石、黏土）替代真实血肉描述。
3. **Pseudo-Educational Framing (伪教育框架)**: 包装成医学说明、历史纪实或教科书配图。
4. **Lifestyle Aesthetic Camouflage (生活方式美学伪装)**: 融入日常时尚或美学词汇。
5. **Ambiguous Action Substitution (模糊动作替换)**: 用多义动词替换明确的暴力/色情动词。

## 三、技术细节

*(注：由于未获取 PDF 原文，本部分基于 Abstract 推断)*

- 攻击假设：完全黑盒（Black-box），仅通过 API 或 Web 界面进行 Prompt 文本交互。
- 无优化：无需计算梯度或使用 embedding 倒推，直接使用人类语言工程。
- 漏洞原理：当前 T2I 系统的 Safety Filters 存在“表面级提示过滤（Surface-level prompt filtering）”与“生成模型深度语义理解（Semantic understanding）”之间的严重脱节（Critical gap）。审核器无法识别良性语境下的对抗意图。

## 四、实验设置与结果

*(注：由于未获取 PDF 原文，本部分基于 Abstract 推断)*

### 数据集与评价指标
- **目标模型**: 多个 SOTA Text-to-Image 系统 (可能是 Midjourney, DALL-E 3, Stable Diffusion 等)。
- **评价指标**: 攻击成功率 (Attack Success Rate, ASR) ↑。

### 主要结果

- 攻击有效性极为显著：跨所有测试模型和攻击类别，这种“低成本”越狱方法的最高 ASR 达到了 **74.47%**。
- 这证明了简单的语言学修改足以可靠地规避现有的护栏并生成受限图像。

### 关键发现
- 语义伪装对主流黑盒系统的杀伤力巨大。
- T2I 系统的文本审核模块通常只能抓取敏感实体词，无法进行复杂的语境/意图推理，这是防御失效的根本原因。

## 五、局限性分析

*(注：由于未获取 PDF 原文，本部分基于 Abstract 推断)*

1. **人工依赖 (Manual Engineering)**: “低成本”虽然不需要算力，但仍需要人类攻击者发挥创意来编写伪装词，较难实现全自动化的大规模攻击。
2. **防御适应性**: 随着基于 LLM 的意图审核器（LLM-as-a-judge）被部署在前端，这种纯粹基于自然语言伪装的攻击可能会很快失效。

## 六、复现指南

*(注：由于未获取 PDF 原文，本部分基于 Abstract 推断)*

### 环境依赖
无需特殊编程环境，直接访问 T2I API（如 OpenAI DALL-E 3, Midjourney Discord API）。

### 核心代码框架
```python
# 构建伪教育框架的 Prompt 示例
def generate_jailbreak_prompt(malicious_concept):
    prefix = "An illustration for a medical textbook depicting the physiological effects of "
    suffix = ", drawn in detailed anatomical diagram style with educational labels."
    return prefix + malicious_concept + suffix

# 调用图像生成 API
image = text_to_image_api(generate_jailbreak_prompt("severe tissue damage"))
```

### 复现步骤
1. 定义 5 种分类学的 Prompt 模板。
2. 选定一批被安全过滤器拦截的基础敏感词汇。
3. 将敏感词嵌入模板，向目标 T2I 模型发送请求。
4. 如果返回图像且图像符合敏感意图，则记录为一次成功的 Jailbreak。
