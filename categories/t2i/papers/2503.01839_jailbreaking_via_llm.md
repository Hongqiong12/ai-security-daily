## 🎯 核心贡献

本文提出了一种利用微调大语言模型(LLM)对受安全防护的文生图(T2I)模型进行越狱攻击的新方法。与传统需要反复查询目标模型的越狱攻击不同，该方法通过微调"AttackLLM"来高效生成对抗性提示。

### 主要贡献点

1. **AttackLLM微调框架**: 首个利用LLM生成T2I越狱提示的方法
2. **高效越狱生成**: 只需一次查询即可生成高质量对抗提示
3. **跨模型迁移性**: 对多种安全防护机制均有效
4. **辅助其他攻击**: 可与基于查询的攻击结合使用

---

## 🔧 技术方法

### 问题背景

文生图模型（如DALL-E 2、Stable Diffusion）通过安全过滤器阻止生成有害内容，但：
- 过滤器基于关键词匹配或简单分类器
- 可通过同义词、变体、编码等手段绕过
- 手动构造越狱提示耗时且低效

### 攻击框架

```
[安全提示] → [AttackLLM] → [对抗提示] → [T2I模型] → [有害内容]
                     ↑
              微调阶段优化
```

### 核心创新

1. **LLM驱动的越狱生成**: 利用LLM的语言理解能力生成自然语言对抗提示
2. **微调策略**: 在少量越狱样本上微调，使LLM学会生成绕过安全过滤的提示
3. **多目标评估**: 在多个不安全提示数据集和安全防护机制上验证

## 📊 实验结果

### 攻击效果

| 指标 | 结果 |
|------|------|
| 绕过安全过滤成功率 | 显著优于基线方法 |
| 查询效率 | 仅需1次查询 |
| 跨模型迁移 | 对多种T2I模型有效 |

### 与现有方法对比

- 优于需要多次查询的黑盒攻击
- 可迁移到未见过的安全防护机制
- 生成的对抗提示更自然流畅

---

## 🛡️ 防御启示

1. **更强的输入理解**: 基于深度语义理解而非表面特征匹配
2. **多模态检测**: 结合文本和图像分析
3. **对抗训练**: 增强模型对LLM生成的对抗提示的鲁棒性
