## 一、背景动机

### 1.1 研究问题

生成式图像建模技术的快速发展带来了严峻的责任部署挑战：

1. **虚假内容泛滥**：AI 生成的逼真图像可能被用于虚假新闻、诈骗
2. **版权归属不清**：难以确定 AI 生成图像的原创者和版权状态
3. **溯源能力缺失**：无法追溯图像的生成来源
4. **恶意使用风险**：深度伪造、身份冒用等

被动检测方法（如 CLIP 分类器）存在局限性，需要主动的水印策略。

### 1.2 核心挑战

1. **水印鲁棒性**：图像经过各种处理（压缩、裁剪、滤镜）后仍能检测
2. **水印隐蔽性**：人类视觉系统无法察觉水印存在
3. **生成效率影响**：水印嵌入不应显著降低图像生成速度
4. **高检测准确率**：极低的误报率和漏报率

### 1.3 核心贡献

Stable Signature 由 Meta AI 提出，是一种**结合图像水印和潜在扩散模型**的主动策略：

> **通过快速微调图像生成器的潜在解码器，使所有生成的图像都携带不可见的二进制签名，以便未来进行检测和溯源。**

核心贡献：

1. **端到端水印方案**：从训练到检测的完整解决方案
2. **潜在空间嵌入**：在扩散模型的潜在空间而非像素空间嵌入水印
3. **极高鲁棒性**：即使裁剪到 10% 仍能 90%+ 准确率检测
4. **极低误报率**：误报率低于 10⁻⁶

## 二、核心方法

### 2.1 方法概述

Stable Signature 的核心思想是**将水印信息编码为扩散模型潜在解码器的条件**，使得生成的每张图像都自动携带可提取的签名。

### 2.2 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                  Stable Signature 架构                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 水印生成器 (Watermark Generator)                          │   │
│  │   - 生成 m 位二进制签名 s ∈ {0,1}^m                      │   │
│  │   - 每个签名对应唯一的模型配置                             │   │
│  └─────────────────────────────────────────────────────────┘   │
│                          │                                      │
│                          ▼                                      │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 潜在解码器微调 (Latent Decoder Fine-tuning)             │   │
│  │   - 基于签名 s 条件化潜在解码器                           │   │
│  │   - 快速度微调 (约 30 分钟)                              │   │
│  │   - 确保每种签名生成图像携带对应水印                      │   │
│  └─────────────────────────────────────────────────────────┘   │
│                          │                                      │
│                          ▼                                      │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 水印提取器 (Watermark Extractor)                        │   │
│  │   - 预训练的盲水印提取网络                               │   │
│  │   - 从图像中恢复隐藏签名                                 │   │
│  │   - 统计检验判断图像来源                                 │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 关键算法

**签名条件化**：

给定二进制签名 $s = (s_1, s_2, ..., s_m)$，通过以下方式条件化解码器：

1. **签名嵌入**：将每个比特 $s_i$ 映射为嵌入向量 $\mathbf{e}_i$
2. **多层注入**：在解码器的多个残差层中注入签名信息
3. **特征调制**：使用自适应实例归一化（AdaIN）调制特征

### 3.2 水印提取器训练

使用对比学习训练盲水印提取器：

```python
# 水印提取器损失函数
def extractor_loss(extractor, images, signatures):
    # 正样本：同签名图像对
    pos_sim = cosine_similarity(extractor(img1), extractor(img2))
    
    # 负样本：不同签名图像对
    neg_sim = cosine_similarity(extractor(img1), extractor(img3))
    
    # 对比损失
    loss = -log(σ(pos_sim)) - log(1 - σ(neg_sim))
    return loss
```

## 四、数学原理

### 4.1 水印编码

设 $s \in \{0,1\}^m$ 为目标签名，$\theta_d$ 为潜在解码器参数。微调目标为：

$$\min_{\theta_d} \mathbb{E}_{z, c, s}[\|x - G_\theta(z, c, s)\|^2]$$

其中 $G$ 是生成器，$z$ 是潜在编码，$c$ 是文本条件。

### 4.2 签名条件化

使用 FiLM（Feature-wise Linear Modulation）层实现签名条件化：

$$\text{FiLM}(h; s) = \gamma(s) \cdot h + \beta(s)$$

其中 $\gamma(s), \beta(s)$ 是从签名 $s$ 推导的调制参数。

### 4.3 检测统计

给定图像 $I$，提取签名 $\hat{s} = E(I)$。通过假设检验判断来源：

$$H_0: I \sim \mathcal{M}_s \quad \text{vs} \quad H_1: I \not\sim \mathcal{M}_s$$

使用似然比检验：

$$\Lambda(I) = \frac{P(I | H_0)}{P(I | H_1)}$$

## 六、复现指南

### 6.1 环境配置

```bash
git clone https://github.com/facebookresearch/stable_signature
cd stable_signature

pip install -r requirements.txt
```

### 6.2 使用示例

```python
from stable_signature import StableSignatureWatermarker

# 初始化带水印的模型
watermarker = StableSignatureWatermarker.from_pretrained(
    "facebook/stable-diffusion-v1-5-watermarked",
    signature_id="model_001"  # 分配唯一的模型签名
)

# 生成带水印图像
images = watermarker.generate(
    prompts=["a beautiful sunset over mountains"],
    num_inference_steps=50
)

# 检测图像来源
for img in images:
    result = watermarker.detect(img)
    print(f"Signature: {result.signature}")  # "model_001"
    print(f"Confidence: {result.confidence:.4f}")  # 0.9987
```

## 七、总结与展望

### 7.1 主要贡献

Stable Signature 是首个针对扩散模型的端到端水印方案：

1. **潜在空间嵌入**：更鲁棒的隐藏水印
2. **极高鲁棒性**：各种攻击下保持高检测率
3. **极低误报**：统计保证的安全性
4. **Meta 开源**：公开可用实现

### 7.2 局限性

1. 需要预知模型签名
2. 检测器需要专门训练
3. 无法防止知识蒸馏攻击

### 7.3 未来方向

1. 多模态水印（视频、音频）
2. 自适应水印强度
3. 对抗鲁棒性增强
