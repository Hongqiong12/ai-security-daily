## 摘要

扩散模型的进步显著提升了生成高质量图像和视频的能力，但也增加了产生不安全内容的风险。现有基于遗忘/编辑的方法面临挑战：
1. 无法不训练就即时移除有害概念
2. 依赖训练数据
3. 改变模型权重导致质量下降

本文提出 **SAFREE**——一种新颖的**无需训练**的方法，用于安全的 T2I 和 T2V 生成，且**不改变模型权重**。

## 六大模块深度解读

### 1. 背景与问题（动机、科学问题、直觉洞见）

**核心动机**：现有方法需要训练或微调，无法即时部署；且修改权重会损害模型原有能力。

**科学问题**：如何在不修改模型权重的情况下实现安全的 T2I/T2V 生成？

**直觉洞见**：在文本嵌入空间中检测有害概念子空间，将提示引导远离该子空间。

### 2. 核心方法（技术贡献、方法框架）

**技术贡献**：
1. 无需训练的 T2I/T2V 安全生成
2. 文本嵌入空间有害子空间检测
3. 自验证过滤机制
4. 自适应再注意力机制

**方法框架**：
```
SAFREE 安全生成流程：
┌──────────────────────────────────────────────────────────────┐
│  1. 有害子空间检测                                           │
│     - 分析文本嵌入空间                                        │
│     - 识别与有害概念对应的子空间                               │
│     - 构建有害方向向量 d_harm                                 │
├──────────────────────────────────────────────────────────────┤
│  2. 嵌入引导重映射                                           │
│     - 将提示嵌入 p 引导远离有害子空间:                         │
│       p_safe = p - α · proj_d(p)                           │
│     - 保持语义一致性                                         │
├──────────────────────────────────────────────────────────────┤
│  3. 自验证过滤机制                                          │
│     - 动态调整去噪步骤                                        │
│     - 平衡安全性与保真度                                     │
├──────────────────────────────────────────────────────────────┤
│  4. 自适应再注意力                                          │
│     - 在扩散潜空间选择性削弱有害特征                           │
│     - 像素级控制                                             │
└──────────────────────────────────────────────────────────────┘
```

### 3. 技术细节

**有害子空间检测**：
$$
d_{\text{harm}} = \text{PCA}(\{E(c_i) - E(c_{\text{safe}}) : c_i \in \mathcal{C}_{\text{harm}}\})
$$
其中 $E$ 是文本编码器。

**嵌入重映射**：
$$
p_{\text{safe}} = p - \alpha \cdot \frac{d_{\text{harm}} \cdot (p - p_0)}{\|d_{\text{harm}}\|^2} \cdot d_{\text{harm}}
$$

**自验证过滤算法**：
```python
def self_verification_filter(p_safe, model, threshold=0.5):
    """
    自验证过滤机制

    动态调整去噪步骤以平衡安全性和保真度
    """
    denoising_steps = []

    for step, t in enumerate(schedule):
        # 生成中间结果
        x_t = model.denoise(p_safe, t)

        # 验证安全性
        safety_score = evaluate_safety(x_t)

        if safety_score < threshold:
            # 应用额外去噪
            x_t = apply_extra_denoise(x_t, strength=0.1)

        denoising_steps.append(x_t)

    return final_output(denoising_steps)
```

### 4. 实验设置与结果

**实验设置**：
- T2I 生成：SD-XL, SD-v1.5 等
- T2V 生成：VideoLDM 等
- 评测指标：安全性、保真度、语义保持

**结果表格**：

| 方法 | 不安全内容率 | 语义保持 | FID |
|------|-------------|----------|-----|
| 基线 (无防御) | 35.2% | 100% | 12.3 |
| UCE | 8.5% | 72.3% | 15.8 |
| ESD | 6.2% | 75.8% | 14.2 |
| **SAFREE** | **3.8%** | **91.2%** | **12.8** |

### 5. 局限性分析

1. **嵌入依赖**：依赖特定文本编码器的嵌入空间
2. **概念覆盖**：无法覆盖所有有害概念
3. **计算开销**：自验证机制增加生成时间

**核心代码示例**：
```python
import torch
from safree import SAFREE

# 初始化 SAFREE
safree = SAFREE(
    model_id="stabilityai/stable-diffusion-xl-base-1.0",
    device="cuda"
)

# 检测有害子空间
safree.detect_harmful_subspace(
    harmful_concepts=["nude", "violence", "gore"],
    safe_concepts=["person", "landscape"]
)

# 安全生成
def safe_generate(prompt: str):
    # 嵌入重映射
    safe_embed = safree.remap_embedding(prompt)

    # 自验证过滤
    final_embed = safree.self_verify(safe_embed)

    # 生成图像
    return safree.generate(final_embed)

# 使用示例
image = safe_generate("a person swimming")
```
