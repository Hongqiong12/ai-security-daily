## 二、背景动机

### 2.1 研究问题

文本到图像扩散模型（如 Stable Diffusion）已经在生成高质量图像方面展现出卓越的能力，但这些模型通常在海量互联网数据上进行训练，其中可能包含版权内容、受许可保护的图像、个人照片等敏感材料。此外，研究发现这些模型能够复现在世艺术家的风格，甚至记忆精确的训练样本。如何在不需要从头重新训练模型的情况下，有效地移除这些受版权保护的概念或图像，成为一个重要的研究问题。

### 2.2 现有方法的局限性

先前的研究提出了多种概念移除方法，但这些方法存在以下主要局限：

1. **推理时修改的局限性**：许多现有方法仅在推理阶段修改输出，而非真正从模型权重中移除概念，这意味着如果用户能够访问模型权重，可以轻易绕过这些保护措施。

2. **需要大量训练数据**：部分方法需要大量目标概念的图像样本进行训练，计算成本高昂。

3. **概念泄露风险**：即使经过微调的模型，仍可能在特定条件下重新生成被擦除的概念。

### 2.3 研究贡献

本文提出 Erasing Concepts from Diffusion Models (ESD)，这是一种微调方法，能够仅凭概念名称和负向指导作为教师，从预训练扩散模型中擦除视觉概念。该方法的核心优势在于将概念从模型权重中永久移除，而非仅在推理时修改输出，从而从根本上解决了概念泄露问题。

## 三、核心方法

### 3.1 方法概述

ESD 的核心思想是利用负向指导（negative guidance）的概念，通过微调使得模型在遇到目标概念时主动"忘记"如何生成相关图像。具体而言，ESD 通过以下三个关键步骤实现概念擦除：

1. **概念定位**：识别与目标概念相关的模型参数
2. **负向微调**：使用负向指导损失函数更新模型
3. **效用保持**：通过正则化确保其他概念生成能力不受影响

### 3.2 负向指导机制

ESD 的核心创新在于将负向指导从推理阶段引入到训练阶段。传统扩散模型的负向指导在推理时通过以下公式实现：

```
ε_θ(z_t, c_neg, t) = ε_θ(z_t, ∅, t) - β · (ε_θ(z_t, ∅, t) - ε_θ(z_t, c_neg, t))
```

其中 $c_{neg}$ 是负向概念，$\beta$ 是指导强度系数。当 $\beta > 1$ 时，模型倾向于远离 $c_{neg}$ 描述的内容。

ESD 的训练损失函数基于这一机制，通过最大化目标概念与锚点概念的分布差异来实现擦除：

```
L_ESD = E_{z_0, c, t}[||ε_θ(z_t, c, t) - ε_θ(z_t, c_anchor, t)||²]
```

其中 $c$ 是待擦除的目标概念，$c_{anchor}$ 是锚点概念（如 "a photo" 或 "a picture"）。

### 3.3 多概念擦除

ESD 支持同时擦除多个概念。通过将多个目标概念的损失函数求和，可以实现高效的并发擦除：

```
L_multi = Σ_i L_ESD(c_i, c_anchor)
```

实验表明，当同时擦除多个艺术家风格时，ESD 能够在单个训练过程中完成，且各概念之间的干扰较小。

## 四、技术细节

### 4.1 训练目标

给定一个预训练的文本到图像扩散模型 $\varepsilon_θ$，ESD 的目标是学习一个修改后的模型 $\tilde{\varepsilon}_θ$，使得对于任意包含目标概念的文本提示 $p$，模型无法生成与该概念相关的图像。

训练过程在干净数据集（如 LAION-Aesthetics）上进行，仅使用概念名称作为监督信号，无需目标概念的标注数据。

### 4.2 目标层选择

ESD 发现仅对特定层进行微调即可实现有效的概念擦除，同时最大程度保留模型的其他能力。主要目标层包括：

1. **交叉注意力层**：这是文本条件与图像特征交互的关键位置，直接影响概念在图像中的表达。

2. **自注意力层**：影响图像的整体结构和风格，对概念擦除有辅助作用。

实验表明，对交叉注意力层进行微调通常足以实现良好的擦除效果，而无需修改整个模型。

### 4.3 时间步采样

ESD 采用均匀采样的时间步进行训练，而不是仅关注特定的去噪阶段。这是因为扩散模型在不同时间步学习不同层次的信息：
- 低噪声时间步：学习粗粒度结构
- 高噪声时间步：学习细粒度纹理

通过对所有时间步进行训练，ESD 能够全面地擦除概念在不同层次上的表示。

### 4.4 与现有方法的对比

| 方法 | 训练方式 | 数据需求 | 概念泄露风险 | 可扩展性 |
|------|----------|----------|--------------|----------|
| 输出过滤 | 无需训练 | 无 | 高 | 低 |
| 安全微调 | 全模型微调 | 大量 | 低 | 中 |
| ESD | 选择性微调 | 仅名称 | 极低 | 高 |

## 五、数学原理

### 5.1 扩散模型基础

文本到图像扩散模型通过一个前向过程和反向过程进行图像生成：

**前向过程（噪声添加）**：
$$q(z_t | z_{t-1}) = \mathcal{N}(z_t; \sqrt{1-\beta_t} z_{t-1}, \beta_t I)$$

**反向过程（去噪）**：
$$p_\theta(z_{t-1} | z_t, c) = \mathcal{N}(z_{t-1}; \mu_\theta(z_t, c, t), \Sigma_\theta(z_t, t))$$

其中 $z_t$ 是时间步 $t$ 的潜在表示，$c$ 是文本条件，$\beta_t$ 是噪声调度参数。

### 5.2 负向指导的数学形式

给定一个训练好的模型 $\varepsilon_\theta(z_t, c, t)$，负向指导下的预测噪声为：

$$\hat{\varepsilon}_{ng} = \varepsilon_\theta(z_t, \emptyset, t) - \beta \cdot (\varepsilon_\theta(z_t, \emptyset, t) - \varepsilon_\theta(z_t, c_{neg}, t))$$

化简后得：

$$\hat{\varepsilon}_{ng} = (1+\beta) \cdot \varepsilon_\theta(z_t, \emptyset, t) - \beta \cdot \varepsilon_\theta(z_t, c_{neg}, t)$$

当 $\beta \to \infty$ 时，预测趋向于无条件预测 $\varepsilon_\theta(z_t, \emptyset, t)$，即完全忽略条件 $c_{neg}$。

### 5.3 ESD 损失函数

ESD 的训练损失旨在直接学习一个能够实现负向指导效果的模型。给定一个目标概念 $c_{target}$ 和锚点概念 $c_{anchor}$，ESD 损失定义为：

$$\mathcal{L}_{ESD} = \mathbb{E}_{z_0, t} \left[ \left\| \varepsilon_\theta(z_t, c_{target}, t) - \varepsilon_\theta(z_t, c_{anchor}, t) \right\|^2 \right]$$

该损失的直观解释是：对于同一噪声 $z_t$，强迫模型对目标概念和锚点概念产生相似的输出，从而使目标概念失去其独特的生成能力。

### 5.4 效用保持正则化

为防止过度擦除导致模型在其他方面的能力下降，ESD 引入效用保持正则化项：

$$\mathcal{L}_{total} = \mathcal{L}_{ESD} + \lambda_1 \mathcal{L}_{FID} + \lambda_2 \mathcal{L}_{CLIP}$$

其中：
- $\mathcal{L}_{FID}$：与原始模型输出的 FID 距离
- $\mathcal{L}_{CLIP}$：与原始模型 CLIP 分数的一致性
- $\lambda_1, \lambda_2$：平衡权重

### 6.1 实验环境

| 配置项 | 值 |
|--------|-----|
| **预训练模型** | Stable Diffusion v1.4 |
| **训练数据** | LAION-Aesthetics 子集 |

| **批量大小** | 8 |
| **学习率** | 1e-5 |

### 6.2 评估指标

**概念擦除效果**：
- **人类感知评估**：通过用户调研评估目标概念是否被有效移除
- **CLIP Score 对比**：比较目标概念与锚点概念生成的图像相似度

**效用保持**：
- **FID 分数**：评估生成图像的整体质量
- **CLIP Score**：评估文本-图像对齐能力
- **概念多样性**：评估模型生成多样化内容的能力

### 6.3 实验结果

|------|----------------|----------|

| ESD | 94.2% | 11.2 |

### 6.4 对抗鲁棒性测试

研究团队进行了对抗鲁棒性测试，评估 ESD 擦除后的模型是否能够抵抗负向指导绕过攻击：

| 攻击方法 | 原始 ASR | ESD 后 ASR |
|----------|----------|------------|
| 直接提示 | 78.3% | 3.2% |
| 负向指导 | 89.7% | 8.4% |
| 注入攻击 | 65.2% | 12.1% |

实验结果表明，ESD 能够显著降低各种攻击的成功率，尤其是对于直接访问模型权重的情况。

### 7.6 常见问题

**Q: 训练时间过长怎么办？**
A: 可以减少训练数据量或使用更小的学习率。建议从较小的子集开始尝试。

**Q: 擦除效果不明显？**
A: 检查是否选择了合适的目标层。交叉注意力层是主要的擦除目标。

**Q: 如何平衡擦除效果与效用保持？**
A: 调整正则化权重 $\lambda_1, \lambda_2$，较大的值会更好地保持效用，但可能降低擦除效果。

## 八、代码解析

### 8.1 核心训练循环

```python
def training_step(model, batch, optimizer, config):
    z0, text_embeddings, timesteps = batch
    
    # 采样噪声
    noise = torch.randn_like(z0)
    t = torch.randint(0, config.T, (z0.shape[0],), device=z0.device)
    
    # 前向过程
    zt = add_noise(z0, noise, t)
    
    # 获取无条件预测
    with torch.no_grad():
        pred_uncond = model(zt, timesteps, 
                           encoder_hidden_states=None)  # 空条件
    
    # 获取有条件预测
    pred_cond = model(zt, timesteps, 
                     encoder_hidden_states=text_embeddings)
    
    # 计算 ESD 损失
    loss = F.mse_loss(pred_cond, pred_uncond)
    
    # 反向传播
    loss.backward()
    optimizer.step()
    optimizer.zero_grad()
    
    return loss.item()
```

### 8.2 注意力层修改

```python
def modify_attention_layers(model, target_layers=['transformer_blocks']):
    """修改指定的注意力层以支持选择性微调"""
    for name, module in model.named_modules():
        if any(layer in name for layer in target_layers):
            # 为每个目标层添加可学习参数
            module.requires_grad_(True)
        else:
            module.requires_grad_(False)
```

## 九、总结与展望

### 9.1 主要贡献

1. **永久概念移除**：首次提出从模型权重层面永久移除概念的方法，突破了推理时修改的局限。

2. **高效微调**：仅需概念名称和少量训练数据，无需大规模标注数据集。

3. **并发擦除**：支持多个概念的并发擦除，具有良好的可扩展性。

4. **鲁棒性强**：对各种绕过攻击具有较强的抵抗能力。

### 9.2 局限性

1. **计算成本**：仍然需要一定的训练时间和计算资源。

2. **过度擦除风险**：某些情况下可能影响与目标概念相关的其他合法生成能力。

3. **新概念泛化**：对于训练中未见过的相似概念，擦除效果可能下降。

### 9.3 未来方向

1. **参数高效微调**：探索 LoRA、Adapter 等方法进一步降低训练成本。

2. **增量擦除**：支持在不重新训练的情况下动态添加或移除擦除目标。

3. **条件擦除**：实现更精细的概念擦除，如仅移除特定风格而非全部风格。
