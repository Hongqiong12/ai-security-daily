## 一、背景与问题

文本到图像（Text-to-Image, T2I）扩散模型（如 Stable Diffusion、DALL-E、Midjourney）在过去两年间取得了令人瞩目的生成质量飞跃。然而，这种强大的图像生成能力也带来了严重的滥用风险——恶意用户可能利用这些模型生成名人色情图像、品牌商标侵权内容、暴力画面或其他有害视觉材料。尽管训练层面的安全措施可以在一定程度上缓解问题，但模型的权重中仍然编码着关于这些概念的知识，一旦绕过训练限制即可被重新激活。

概念擦除（Concept Erasure）技术的目标是：在不重新训练整个扩散模型的前提下，通过对模型进行轻量级修改，将特定概念从模型的生成能力中彻底移除，同时尽可能不影响模型对其他无关概念的生成质量。这是一个看似简单、实则困难的多目标优化问题——擦除必须足够彻底（有效性），但不能以牺牲模型整体效能为代价（保真度）。

**现有方法的根本局限**：当前主流概念擦除方法（包括 ESD、UCE、MEMIT、MACE 等）几乎都聚焦于修改扩散模型的**交叉注意力层（Cross-Attention Layers）**。这些层负责将文本条件嵌入到图像生成过程中，是文本到视觉概念映射的核心通道。论文的作者首先从数学上揭示了一个关键问题：**仅更新交叉注意力层的权重，在数学上等价于向模型添加线性模块**。这意味着现有方法本质上都是在做线性操作——它们能够调整概念在"该出现/不该出现"上的开关状态，却难以处理概念之间的非线性交互关系。

具体来说，名人 A 和名人 B 在视觉特征空间中可能共享某些底层模式（如光照、姿态）。当线性方法试图"关闭"名人 A 时，这种线性调整会不可避免地波及那些共享底层模式的其他概念（包括普通人物），导致模型的生成多样性大幅下降——这就是"灾难性遗忘"（Catastrophic Forgetting）问题在概念擦除领域的体现。

**核心科学问题**：如何在保留模型非线性表征能力的同时，实现对目标概念的精准擦除，并确保剩余概念的完整保留？

直觉洞见：类比医学手术，线性方法相当于用一把大锤调整基因表达——粗暴且副作用大；非线性门控机制则相当于为每个概念配备一把独立的手术刀，仅切除目标组织而不影响周围健康组织。ResAG（残差注意力门）正是这把"手术刀"的实现。

## 二、核心方法

CPE 提出了一个系统性的三层解决方案，分别针对概念定位、概念擦除和鲁棒性增强：

### 2.1 残差注意力门（Residual Attention Gate, ResAG）

这是 CPE 框架的核心创新。ResAG 被嵌入到扩散模型 U-Net 的交叉注意力层中，作为一个可学习的非线性门控模块。

**与线性方法的本质区别**：

- **传统线性方法**：对交叉注意力层的 Query、Key、Value 矩阵进行线性微调，数学上相当于 $W' = W + \Delta W$，本质是对注意力分数做线性变换
- **CPE 非线性门控**：引入非线性激活函数 $\sigma(\cdot)$，通过门控机制动态调节注意力分布：
  $$Attention_{CPE}(Q, K, V) = Attention(Q, K, V) \odot G(Q, K)$$
  其中 $G(Q, K) = \sigma(W_g \cdot [Q; K])$ 是残差注意力门，$\odot$ 为逐元素乘法，$[;]$ 为拼接操作

**ResAG 的设计哲学**：门控机制允许模型独立地"关闭"与目标概念相关的注意力路径，而不会对其他路径造成线性干扰。这类似于神经科学中的"选择性神经抑制"——大脑可以选择性地抑制特定神经通路而不影响整体神经网络的功能。

### 2.2 注意力锚定损失（Attention Anchoring Loss）

为了解决擦除过程中的灾难性遗忘问题，CPE 设计了一种注意力锚定损失函数：

$$L_{anchor} = \sum_{l \in \mathcal{L}} D_{KL}(A^l_{baseline} \| A^l_{current})$$

其中 $A^l$ 是第 $l$ 层交叉注意力图（attention map），$D_{KL}$ 是 KL 散度，$\mathcal{L}$ 是目标注意力层集合。

**物理含义**：约束模型在擦除目标概念后的注意力分布，与原始未修改模型（baseline）的注意力分布尽可能相似。这确保了擦除操作是"局部"的——只影响目标概念相关的注意力路径，而不改变其他概念的表示空间分布。

### 2.3 对抗训练与可学习文本嵌入

论文观察到，擦除后的模型仍然容易受到**对抗性文本提示**（adversarial text prompts）的攻击——攻击者通过精心设计的文本（如添加特殊字符、拼写变形）绕过概念检测。

CPE 的解决方案是：
1. **可学习文本嵌入（Learnable Text Embeddings）**：将目标概念的文本嵌入作为可学习参数，在训练过程中不断更新，使模型学会识别概念的多种文本表达变体
2. **对抗性训练迭代**：在每次训练迭代中，自动生成针对当前模型的对抗性文本提示（使用梯度扰动或随机搜索），并将它们纳入训练数据

训练流程的完整迭代步骤：

```
for each concept c to erase:
    Initialize ResAG_g with random weights
    Initialize learnable text embedding T_c for concept c

    for epoch in range(N_epochs):
        # Step 1: Forward pass with ResAG
        attention_maps = U-Net_forward(image, T_c, ResAG)

        # Step 2: Compute erasing loss (reduce concept activation)
        L_erase = CE(concept_detector(predicted_image), target=erased)

        # Step 3: Compute anchoring loss (preserve other concepts)
        L_anchor = KL(attention_maps_current || attention_maps_baseline)

        # Step 4: Adversarial text generation
        T_adv = generate_adversarial_prompts(T_c, model)

        # Step 5: Total loss and update
        L_total = L_erase + λ_anchor * L_anchor + λ_adv * L_erase(T_adv)
        update(ResAG, T_c) via gradient descent
```

## 三、技术细节

### 3.1 交叉注意力层在扩散模型中的角色

在 Stable Diffusion 等主流 T2I 模型中，文本条件通过以下机制影响图像生成：

1. **文本编码器**（如 CLIP Text Encoder）将输入文本 $t$ 编码为条件嵌入 $c = \text{TextEncoder}(t)$
2. **交叉注意力层**将图像 latent 表示 $z$ 与文本嵌入 $c$ 进行交叉注意计算：
   $$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d}}\right) V$$
   其中 $Q = W_Q \cdot z$, $K = W_K \cdot c$, $V = W_V \cdot c$
3. **U-Net** 通过自注意力和交叉注意力迭代去噪，生成最终图像

概念（如"Scarlett Johansson"）在模型中通过**注意力分布模式**体现：当用户输入包含该概念的文本时，模型会在图像的相关区域激活特定的注意力头（attention heads），引导生成对应的视觉特征。

### 3.2 ResAG 的数学形式

在原始交叉注意力中引入 ResAG 后：

$$\tilde{A} = \text{Attention}(Q, K, V)$$

$$A_{gate} = \sigma(W_2 \cdot \text{GELU}(W_1 \cdot [Q; K]))$$

$$A_{final} = \tilde{A} \odot (1 - A_{gate}) + \tilde{A}_{target} \odot A_{gate}$$

其中第二项的物理含义是：当 $A_{gate}$ 接近 1 时，注意力被重新路由到目标概念的参考注意力图 $\tilde{A}_{target}$，实现概念替换而非简单删除（这比直接置零更平滑，避免注意力分布出现突变）。

### 3.3 多概念合并机制

在实际应用中，用户往往需要同时擦除多个概念（如同时删除多个名人的生成能力）。CPE 通过以下方式实现多概念合并：

$$A_{final} = \sum_{i} w_i \cdot A^i_{gate}$$

其中 $w_i = \frac{\exp(g_i)}{\sum_j \exp(g_j)}$ 是通过 softmax 归一化的门控权重，$g_i$ 是第 $i$ 个概念的未归一化门控值。

**优势**：推理时只需加载所有概念的预训练 ResAG，通过加权求和即可合并，无需重新训练。

## 四、实验设置与结果

### 4.1 概念类别与数据集

实验覆盖三类最具代表性的擦除目标：

| 概念类别 | 代表样本 | 擦除挑战 |
|:---|:---|:---|
| 名人（Celebrities） | 50 位公众人物 | 需要完全消除面部相似性，同时保持一般人物生成能力 |
| 艺术风格（Artistic Styles） | 100 种绘画风格 | 风格是弥漫性的，难以精确定位 |
| 露骨内容（Explicit Contents） | 4 种敏感类别 | 需要在强约束下仍保持正常内容生成质量 |

### 4.2 评估指标

| 指标 | 含义 | 目标 |
|:---|:---|:---|
| **FID** | Frechet Inception Distance，衡量生成分布与真实分布的差异 | 越低越好（表示生成质量高） |
| **KID** | Kernel Inception Distance，FID 的无偏替代 | 越低越好 |
| **CLIP Score** | 衡量生成图像与文本提示的语义一致性 | 越高越好（表示忠实于文本） |
| **GCD（GIPHY Celebrity Detector）** | 衡量擦除后模型生成名人图像的概率 | 越低越好（表示擦除成功） |
| **NudeNet Score** | 衡量露骨内容检测置信度 | 越低越好（表示擦除成功） |

### 4.3 核心性能对比

论文在 Celebrity100 数据集（包含 50 位名人的图像）上的关键结果：

| 方法 | 概念擦除率（↑） | 剩余概念 FID（↓） | 对抗鲁棒性 |
|:---|:---:|:---:|:---:|
| **CPE（本文）** | **>95%** | **最优** | **强** |
| MACE | ~90% | 较差 | 中等 |
| UCE | ~85% | 中等 | 弱 |
| MEMIT | ~88% | 差 | 弱 |
| ESD | ~80% | 中等 | 弱 |

**关键发现**：
- CPE 在**擦除率**和**剩余概念保真度**两个维度上同时优于所有基线方法
- 对于需要同时擦除多个概念的复合场景，CPE 的性能衰减远小于线性方法
- 对抗性提示攻击实验表明，CPE 对梯度扰动提示和拼写变形攻击均表现出强鲁棒性

### 4.4 消融实验关键发现

| 消融组件 | 对擦除率的影响 | 对 FID 的影响 |
|:---|:---:|:---:|
| 去除 ResAG（非线性），仅用线性微调 | 擦除率下降 **~15%** | FID 上升 |
| 去除注意力锚定损失 | 擦除率不变 | FID 显著上升（灾难性遗忘） |
| 去除对抗训练 | 擦除率不变 | 对抗攻击成功率上升 **~20%** |
| 仅用线性门控（无残差连接） | 性能下降 | 注意力分布出现不连续跳变 |

## 六、与 TINA 攻击的关系：双向博弈

值得注意的是，CPE 所在的**概念擦除领域**与另一篇报告论文 **TINA（Text-Free Inversion Attack，arXiv 2603.17828）**形成了攻防博弈：

- **CPE**（防御方）：提出更精细的非线性概念擦除机制，试图彻底移除概念知识
- **TINA**（攻击方）：揭示即使是最强的概念擦除方法（包括 CPE），仍然可以通过无文本反演攻击恢复被擦除概念

这一博弈关系揭示了当前 T2I 安全的核心困境：**基于输出层的擦除无法真正移除模型权重中编码的视觉知识**。未来真正有效的概念擦除可能需要结合表示层干预（representation-level intervention）和知识蒸馏（knowledge distillation）技术。
