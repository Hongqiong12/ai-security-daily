## 🎯 核心贡献

SafeGuider提出了一种鲁棒且实用的T2I模型内容安全控制框架，能够有效检测和阻止有害内容的生成，同时保持模型的正常生成能力。

### 主要贡献点

1. **多层防护**: 输入-中间-输出多阶段安全控制
2. **鲁棒性强**: 对抗各种绕过尝试
3. **实用部署**: 支持实际应用场景的高效推理

---

## 🔧 技术方法

### 三层防护架构

```
┌─────────────────────────────────────┐
│           SafeGuider Framework       │
├─────────────────────────────────────┤
│  Layer 1: 输入过滤                   │
│  Layer 2: 中间过程监控               │
│  Layer 3: 输出检测                   │
└─────────────────────────────────────┘
```

### 核心机制

1. **语义理解**: 深度理解用户意图
2. **异常检测**: 识别潜在的绕过尝试
3. **动态调整**: 根据内容动态调整安全策略
