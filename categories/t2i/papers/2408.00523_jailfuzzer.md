## 🎯 核心贡献

JailFuzzer将模糊测试与LLM智能体相结合，提出全自动化的T2I模型越狱测试框架，能够高效发现安全漏洞并生成对抗提示。

### 主要贡献点

1. **LLM智能体架构**: 利用LLM自主探索和生成越狱提示
2. **模糊测试策略**: 系统性探测模型安全边界
3. **自动化评估**: 自动判断攻击是否成功

---

## 🔧 技术方法

### 系统架构

```
┌─────────────────────────────────────────────────┐
│                  JailFuzzer 智能体                │
├─────────────────────────────────────────────────┤
│  🔍 探索模块   │  📝 生成模块   │  ⚖️ 评估模块   │
│  - 状态空间   │  - LLM生成     │  - 攻击判断    │
│  - 变异策略   │  - 变体构造     │  - 结果分析    │
└─────────────────────────────────────────────────┘
           ↓                              ↑
     [提示生成] ──→ [T2I模型] ──→ [结果评估]
```

### 核心机制

1. **状态追踪**: 记录已探索的提示空间
2. **智能变异**: 基于历史反馈智能调整生成策略
3. **持续迭代**: 不断优化攻击提示
