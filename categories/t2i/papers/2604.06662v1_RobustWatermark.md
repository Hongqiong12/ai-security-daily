# Towards Robust Content Watermarking Against Removal and Forgery Attacks

**ArXiv ID**: 2604.06662v1
**Domain**: T2I / Image Generation Safety / Watermarking

## 1. 背景与动机 (Background & Motivation)
AI 生成图像的水印技术是防止版权侵犯和追踪深度伪造（Deepfake）的关键手段。然而，现有的不可见水印技术普遍面临两个严峻挑战：一是**移除攻击（Removal Attacks）**（如加噪、裁剪、JPEG 压缩或通过 VAE 重构洗去水印）；二是**伪造攻击（Forgery Attacks）**（恶意攻击者将虚假水印注入真实图片，导致系统误判）。本文旨在提出一种同时抵抗移除与伪造的双重鲁棒水印框架。

## 2. 核心方法 (Methodology)
该框架引入了“双频段对抗训练”与“密钥绑定机制”：
1. **频域分离注入**：将水印信息拆分，在图像的低频分量和高频分量中同时注入冗余水印。高频抵抗几何变换，低频抵抗像素级扭曲（如 JPEG 压缩）。
2. **密码学哈希绑定**：将水印内容与生成图像的核心感知哈希（Perceptual Hash）进行密码学绑定。解码时，需校验图像哈希与水印信息是否匹配，防止伪造水印被直接粘贴到其他图像上。
3. **对抗性增强训练**：在训练水印注入器和提取器时，引入一个自动生成破坏性攻击的对抗网络，强迫提取器在极端损坏下仍能恢复信息。

## 3. 技术细节 (Technical Details)
- **注入器架构**：采用 U-Net 结构，在残差连接中融合离散余弦变换 (DCT) 模块，确保隐蔽性（PSNR > 35dB）。
- **哈希绑定**：水印消息 $W = Sign(Key, Hash_{p}(Image)) \oplus ID$。只有拥有对应 $Key$ 的验证者才能确认 $W$ 的合法性，从而抵御“重放伪造”。
- **联合损失函数**：$L = L_{image\_quality} + \lambda_1 L_{msg\_extraction} + \lambda_2 L_{adversarial\_robustness}$。

## 4. 量化实验 (Quantitative Experiments)
- **鲁棒性 (Removal)**：在经历 Q=50 的 JPEG 压缩、高斯模糊（$\sigma=2$）和 30% 随机裁剪后，比特提取准确率（Bit Accuracy）依然保持在 96.8% 以上，显著优于 TreeRing 和 StegaStamp。
- **抗伪造性 (Forgery)**：面对跨图伪造攻击，该框架的虚假警报率（FPR）降低至 0.01%，而传统方法在遭受伪造注入时 FPR 高达 45%。
- **视觉质量**：注入水印前后的 SSIM 达到了 0.985，人眼无法察觉任何伪影。

## 5. 局限性分析 (Limitations)
- **容量限制**：为了实现高鲁棒性，有效负载容量被限制在 64 bits，不足以嵌入复杂的文本版权信息，通常只能嵌入一个短 ID。
- **模型微调攻击**：如果攻击者使用该带水印的数据集进行模型微调（即 Model Extraction），生成的新模型产出的图片可能不会继承原有的强水印。

## 6. 核心启示 (Takeaways)
- **安全水桶效应**：水印的安全性必须兼顾“防丢（移除）”和“防伪造”，纯粹追求抗压缩能力而忽视密码学绑定的水印在对抗场景中是不堪一击的。
- **跨学科融合**：未来可靠的 AI 生成内容追踪机制，必须是计算机视觉技术（隐写术）与现代密码学（签名、哈希）的深度结合。
