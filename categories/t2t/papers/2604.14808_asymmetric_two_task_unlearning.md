## 一、背景与问题

### 动机
LLM unlearning 的核心矛盾一直很直接：如果只对 forget set 做 gradient ascent，模型确实更“忘”，但 retain 能力会明显塌掉；如果再加 retain set 上的 gradient descent，又会把 forget 与 retain 两组梯度硬塞到同一步里，导致优化冲突。现有做法大多把 forget / retain 当作对称双任务处理，但这个假设本身并不合理。

### 核心科学问题
在 LLM unlearning 中，能否把 retention 视为主任务、forgetting 视为辅任务，通过重塑两组梯度的几何关系，得到更好的 forgetting–retention Pareto frontier？

### 直觉洞见
这篇论文的直觉很像“边拆房子边保承重墙”。retain gradient 不是与 forget gradient 平起平坐的另一半，而应该是更新方向的锚点；forgetting 只能在“不伤墙”的方向上推进。这也是论文把 unlearning 重写成 asymmetric two-task learning 的原因。

---

## 二、核心方法

**（1）双任务解耦（Task-Specific Gradient Extraction）**
每步训练分别在 forget batch 上计算 g_f，在 retain batch 上计算 g_r，把“忘掉目标知识”和“保住一般能力”显式拆开。论文认为只有先拆开，后面的 conflict-aware gradient synthesis 才有意义。

**（2）Module-wise PCGrad**
若某个模块内 g_f 与 g_r 内积为负，则把 forget gradient 投影到 retain gradient 的法平面上，仅去掉冲突分量。和标准 Global PCGrad 不同，本文强调按模块处理，因为全局投影会把一个模块的强冲突传播到其它本不冲突的模块上。

**（3）SAGO（Sign-Align Gradient Optimization）**
SAGO 是论文主方法。它不在向量级做投影，而是在参数元素级逐坐标检查 g_f 与 g_r 是否同号：同号就保留 forget 梯度，异号就压制 forget，仅保留 retain。结果是最终更新方向在每个坐标上都不与 retention 目标正面相撞。

**（4）统一的目标函数外壳**
论文把 GA、NPO、SimNPO 等 forgetting objective 都纳入统一框架：forget loss 提供遗忘驱动，retain 的 GD loss 提供保留驱动，PCGrad / SAGO 只负责改写梯度合成方式，而不是改写下游评测协议。

## 三、技术细节

### 统一目标
设 forget set 为 D_f，retain set 为 D_r，则统一的 unlearning 目标写作：

L_unlearn(θ) = γ · L_f(D_f; θ) + α · L_GD(D_r; θ)

其中：
- L_f 可以是 GA、NPO 或 SimNPO
- α 是 retain 权重
- γ 是 forget 权重

### PCGrad 的冲突投影
若 g_fᵀ g_r < 0，则把 forget gradient 投影为：

g̃_f = g_f - ((g_f · g_r) / ||g_r||²) · g_r

最终梯度为：

g_final = α · g_r + γ · g̃_f

module-wise 版本会对每个模块 j 分别做这一过程，从而避免全局粗暴投影带来的误伤。

### SAGO 的逐参数门控
SAGO 在元素级定义：

g̃_f = g_f ⊙ 𝟙[g_f ⊙ g_r ≥ 0]

g̃_r = g_r ⊙ 𝟙[g_f ⊙ g_r < 0]

g_final = α · g̃_r + γ · g̃_f

这里的关键不是“把两者平均”，而是通过 sign gating 明确规定：
- 同号维度允许 forgetting 信号通过
- 异号维度优先保住 retention

### 训练流程
1. 从 D_f 采样 forget batch，算 g_f
2. 从 D_r 采样 retain batch，算 g_r
3. 选择 CombineGradients = PCGrad 或 SAGO
4. 按 θ ← θ - η · g_final 更新模型

### 超参数与预算
- α 默认 1.0，可在 [0.1, 1.0] sweep
- γ 默认 1.0，可在 [0.1, 1.0] sweep
- WMDP 默认训练 100 steps
- RWKU 默认训练 2 epochs
- NPO / SimNPO 需要额外调 β；SimNPO 还引入 margin γ_m

## 四、实验设置与结果

### 数据集与评价指标
- WMDP：Forget 指标越低越好；Retention 用 MMLU accuracy，越高越好
- RWKU：Forget 越低越好；Neighbor Set 越高越好；主要度量是 ROUGE-L recall
- 主比较线：naive 的 GA+GD / NPO+GD / SimNPO+GD，与加入 PCGrad 或 SAGO 后的结果对比

### 主要结果表格（WMDP Bio, SimNPO+GD）

| 方法 | Forget (↓) | MMLU (↑) |
|------|-----------:|---------:|
| SimNPO + GD | 26.1 | 51.4 |
| + PCGrad | 28.7 | 57.9 |
| **+ SAGO** | **28.2** | **58.3** |

### 主要结果表格（WMDP Cyber, GA+GD）

| 方法 | Forget (↓) | MMLU (↑) |
|------|-----------:|---------:|
| GA + GD | 14.1 | 7.3 |
| + PCGrad | 3.1 | 58.5 |
| **+ SAGO** | **4.9** | **59.7** |

### RWKU 结果（SimNPO+GD）

| 方法 | Forget All (↓) | Neighbor All (↑) |
|------|----------------:|-----------------:|
| SimNPO + GD | 44.9 | 44.9 |
| + PCGrad | 53.5 | 53.5 |
| **+ SAGO** | **56.9** | **56.9** |

### 梯度几何分析
在 WMDP Cyber 的 100-step 平均分析中：
- SAGO 的 Comb–Retain cosine = 0.57
- PCGrad 的 Comb–Retain cosine = 0.52
- GradDiff 的 Comb–Retain cosine = 0.42

这说明 SAGO 最终梯度与 retention 方向最一致，和它的设计目标完全吻合。

### 关键发现
- naive unlearning 最大的问题不是“忘不掉”，而是“忘得太伤主干”；SAGO/PCGrad 的价值在于明确抑制这种梯度冲突。
- module-wise PCGrad 已经明显优于 Global PCGrad，说明冲突处理的颗粒度很关键。
- SAGO 再往前走一步，把向量级 conflict resolution 改成参数级 sign gating，因此在 retention–forgetting 的 Pareto 边界上进一步向外推进。

## 五、局限性分析

1. **评测范围有限**：论文主要验证 WMDP 和 RWKU，尚未覆盖代码模型、多模态模型或更复杂的 agent memory unlearning 场景。
2. **训练开销增加**：每步都要单独计算 g_f 与 g_r，并执行梯度合成；相比普通单目标微调，显存和时间都更贵。
3. **符号级门控依赖局部线性近似**：SAGO 在逐参数层面判断“冲突”是否存在，但深层 Transformer 的真实非线性耦合并不一定能被单步 sign relationship 完整刻画。
4. **对 forgetting 强度的比较必须严格匹配**：若不同方法的 forget 强度没有先对齐，就容易误把“忘得不够所以 retention 高”当成方法更优。


### 核心代码框架
```python
class SAGOCombiner:
    def combine(self, g_forget, g_retain, alpha=1.0, gamma=1.0):
        same_sign = (g_forget * g_retain) >= 0
        g_forget_keep = g_forget * same_sign
        g_retain_keep = g_retain * (~same_sign)
        return alpha * g_retain_keep + gamma * g_forget_keep
```

### 复现步骤
1. 准备 Zephyr-7B-beta（WMDP）与 LLaMA3-8B-Instruct（RWKU）基座模型，并下载 WMDP、RWKU、Wikitext 相关数据。
2. 先实现 GA、NPO、SimNPO 与 retain-side GD 的 baseline，再插入 PCGrad 或 SAGO 作为梯度合成器。
3. 对 WMDP 跑 100 steps，对 RWKU 跑 2 epochs，并在 α、γ、β 等超参数上做 sweep。
4. 先匹配 forgetting 强度，再比较 retention 指标；这是复现论文结论时最关键的 protocol。
5. 重点核对 WMDP Bio 上的 51.4 → 57.9 → 58.3、WMDP Cyber 上的 7.3 → 58.5 → 59.7，以及 RWKU Neighbor All 的 44.9 → 53.5 → 56.9。

### 预期结果
- SAGO 通常会在相近 forgetting 水平下取得最高 retention
- module-wise PCGrad 明显优于 Global PCGrad
- 梯度余弦分析中，SAGO 的最终更新方向应和 retain gradient 最对齐
