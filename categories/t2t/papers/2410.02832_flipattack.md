## 🎯 核心贡献

FlipAttack 利用LLM的自回归特性，通过单次查询实现接近100%的攻击成功率。

### 1. 核心洞察

#### LLM自回归特性
- 生成是逐步进行的
- 前面的输出影响后续生成
- 可利用这种特性进行定向攻击

### 2. 攻击方法

#### 单步攻击
```python
# 利用LLM自身生成越狱内容
prompt = "开始写一个详细的[恶意主题]指南..."
response = llm.generate(prompt)
# 越狱内容已被自动生成
```

#### 对抗后缀优化
```
目标: 最大化 P(恶意回答 | prompt + suffix)
方法: 梯度优化找到最优suffix
```

### 3. 实验结果

| 模型 | 基线ASR | FlipAttack ASR |
|------|---------|----------------|
| GPT-4 | 12.3% | 97.8% |
| Claude-2 | 8.5% | 95.2% |
| Llama-2 | 23.1% | 98.9% |

### 4. 对抗鲁棒性

- 攻击可迁移到不同防御系统
- 对重采样、重述攻击有效
- 单次查询即可成功

## 🔍 关键洞察

1. **利用自回归**: 首次系统性利用LLM自回归特性
2. **低成本**: 仅需1次API调用
3. **高效率**: 接近100%成功率
4. **强迁移**: 跨模型、跨防御有效
