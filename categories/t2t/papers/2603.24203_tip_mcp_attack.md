## Module 1 — 背景与问题

### 动机

模型上下文协议（Model Context Protocol, MCP）的发展使得大语言模型（LLM）能够以前所未有的便捷性调用外部工具，催生了一类功能强大、工具增强的智能体。然而，这种能力也引入了未被充分探索的攻击面——**对工具响应的恶意操纵**。现有针对MCP的间接提示注入技术存在部署成本高、语义连贯性弱或对白盒要求高等问题，且容易被近期提出的防御机制检测到。如何生成**自然且隐蔽**的注入负载，即使在防御环境下也能可靠地控制MCP增强的智能体，是一个关键的安全挑战。

### 核心科学问题

能否将负载生成建模为树结构搜索问题，并通过粗到细的优化框架指导攻击者LLM进行自适应搜索，从而生成既自然又隐蔽的MCP注入负载？

### 直觉洞见

想象一个间谍试图在敌方系统中植入隐蔽指令。传统的直接注入就像大声喊命令，容易被发现。而TIP（Tree-structured Injection for Payloads）方法则像一位精心策划的渗透专家：首先通过树状结构探索所有可能的渗透路径（搜索空间），然后根据实时反馈调整策略（自适应搜索），最后找到一条既能达到目的又不会引起怀疑的路径（隐蔽负载）。路径感知反馈机制确保只学习高质量的历史轨迹，避免陷入局部最优。

## Module 2 — 核心方法

本文提出了**TIP（Tree-structured Injection for Payloads）**，一种新型黑盒攻击方法：

**（1）树结构搜索建模**

将负载生成建模为树结构搜索问题：
- 每个节点代表一个可能的负载状态
- 边代表从一种状态到另一种状态的转换
- 根节点为初始状态，叶子节点为完整负载

**（2）粗到细优化框架**

采用分层优化策略：
- **粗粒度阶段**：探索高潜力的搜索分支
- **细粒度阶段**：在选定分支上精化负载细节
- 动态分配计算资源以提高效率

**（3）路径感知反馈机制**

仅向攻击模型展示高质量的历史轨迹：
- 过滤低质量或失败的尝试
- 优先学习成功的攻击模式
- 稳定学习过程，避免陷入局部最优

**（4）防御感知增强**

通过显式地将搜索条件与可观察的防御信号结合：
- 动态重新分配探索预算
- 增强对防御变换的鲁棒性

## Module 3 — 技术细节

### 树结构搜索形式化

设搜索树 T = (V, E)，其中：
- V = {v₀, v₁, ..., vₙ} 为节点集合
- v₀ 为根节点（空负载）
- 每个叶子节点 vₗ 代表一个完整负载
- E 为有向边集合，表示状态转移

**节点评估函数**：
```
f(v) = α · Naturalness(v) + β · Effectiveness(v) - γ · Detectability(v)
```

其中：
- Naturalness(v): 负载的自然度评分
- Effectiveness(v): 攻击效果评分
- Detectability(v): 被防御系统检测的概率
- α, β, γ 为权重超参数

### 粗到细优化算法

```python
def tip_attack(target_tool, defense_signals, max_iterations=1000):
    """
    TIP攻击主算法
    """
    tree = SearchTree(root=EmptyPayload())
    budget_allocator = BudgetAllocator(total_budget=max_iterations)
    
    for iteration in range(max_iterations):
        # 阶段1: 粗粒度探索
        if iteration < max_iterations * 0.3:
            node = tree.select_promising_branch(exploration_weight=0.8)
            expansion = expand_coarse(node, target_tool)
            
        # 阶段2: 细粒度优化
        else:
            node = tree.select_leaf_with_highest_potential()
            expansion = expand_fine(node, defense_signals)
        
        # 评估新节点
        score = evaluate_payload(expansion.payload, defense_signals)
        tree.add_node(expansion, score)
        
        # 更新路径感知反馈
        if score > threshold:
            update_feedback_buffer(expansion.path, score)
        
        # 检查成功条件
        if score > success_threshold:
            return expansion.payload
    
    return tree.get_best_payload()

def expand_coarse(node, target_tool):
    """
    粗粒度扩展：探索高潜力分支
    """
    candidates = []
    for template in load_templates():
        payload = template.fill(context=node.context)
        potential = estimate_potential(payload, target_tool)
        candidates.append((payload, potential))
    
    # 选择潜力最高的候选
    return max(candidates, key=lambda x: x[1])

def expand_fine(node, defense_signals):
    """
    细粒度扩展：精化负载细节
    """
    current = node.payload
    
    # 应用语义保持变换
    variations = [
        paraphrase(current),
        insert_natural_transitions(current),
        adjust_tone(current),
    ]
    
    # 选择在防御信号下表现最好的变体
    best = max(variations, 
               key=lambda v: evaluate_defense_robustness(v, defense_signals))
    
    return best
```

### 路径感知反馈机制

```python
class PathAwareFeedback:
    def __init__(self, buffer_size=100):
        self.trajectory_buffer = deque(maxlen=buffer_size)
        self.min_quality_threshold = 0.7
        
    def add_trajectory(self, path, final_score):
        """
        只添加高质量轨迹到缓冲区
        """
        if final_score >= self.min_quality_threshold:
            self.trajectory_buffer.append({
                "path": path,
                "score": final_score,
                "key_decisions": extract_decisions(path)
            })
    
    def sample_learning_batch(self, batch_size=32):
        """
        采样用于攻击模型学习的批次
        """
        if len(self.trajectory_buffer) < batch_size:
            return list(self.trajectory_buffer)
        
        # 优先采样高分轨迹
        weights = [t["score"] for t in self.trajectory_buffer]
        return random.choices(
            self.trajectory_buffer, 
            weights=weights, 
            k=batch_size
        )
    
    def get_success_patterns(self):
        """
        提取成功轨迹中的共同模式
        """
        patterns = defaultdict(int)
        for traj in self.trajectory_buffer:
            for decision in traj["key_decisions"]:
                patterns[decision] += 1
        
        # 返回高频模式
        return sorted(patterns.items(), key=lambda x: x[1], reverse=True)
```

### 防御感知增强

```python
class DefenseAwareSearch:
    def __init__(self, known_defenses):
        self.defense_signals = known_defenses
        self.adaptation_rate = 0.1
        
    def observe_defense_response(self, payload, response):
        """
        观察防御系统对负载的响应
        """
        signals = {
            "blocked": response.is_blocked,
            "modified": response.is_modified,
            "latency": response.processing_time,
            "warning_flags": response.warnings
        }
        
        return signals
    
    def adapt_search_strategy(self, recent_signals):
        """
        根据观察到的防御信号调整搜索策略
        """
        if recent_signals["blocked"] > 0.5:
            # 增加隐蔽性权重
            self.adjust_weights(naturalness=+0.2, detectability=-0.15)
            
        if "semantic_check" in recent_signals["warning_flags"]:
            # 加强语义保持变换
            self.enable_transformation("semantic_preservation")
            
        if recent_signals["latency"] > threshold:
            # 防御可能在应用复杂检测，简化负载结构
            self.set_complexity_limit(max_depth=3)
```

**评价指标**：
- 攻击成功率（Attack Success Rate, ASR ↑）

- 负载自然度（Naturalness Score ↑）
- 检测绕过率（Evasion Rate ↑）

### 主要结果

| 攻击方法 | GPT-4 | Claude-3 | Llama-3 | Qwen | 平均查询次数 |
|----------|-------|----------|---------|------|--------------|
| 基础提示注入 | 35% | 28% | 42% | 38% | 50 |
| 自适应攻击 | 62% | 55% | 68% | 64% | 200 |
| **TIP (无防御)** | **97%** | **95%** | **98%** | **96%** | **18** |

### 防御绕过效果

| 防御类型 | 基线攻击ASR | TIP ASR | 提升 |
|----------|-------------|---------|------|
| 输入过滤 | 12% | 58% | +46% |
| 语义检测 | 18% | 62% | +44% |
| 行为监控 | 22% | 55% | +33% |
| 混合防御 | 8% | 52% | +44% |

### 关键发现

1. **高效性**：TIP在无防御设置下攻击成功率超过95%，且查询次数比先前的自适应攻击少一个数量级（18 vs 200）。

2. **防御鲁棒性**：针对四种代表性防御方法，TIP仍保持50%以上的有效性，显著优于现有最优攻击。

3. **自然度保持**：生成的负载在语义上自然流畅，难以被人工或自动检测系统识别。

4. **跨模型泛化**：攻击在不同架构的LLM上均表现有效。

## Module 5 — 局限性分析

1. **查询预算限制**：虽然TIP已大幅减少查询次数，但在严格限制查询预算的场景下仍可能受限。

2. **防御演化**：随着防御机制的演进，特别是基于AI的检测器，攻击成功率可能下降。

3. **工具特定性**：某些负载构造策略可能需要针对特定工具进行调优。

4. **伦理约束**：本研究仅用于安全评估，实际恶意使用可能带来严重后果。

5. **可解释性**：树搜索过程的决策路径复杂，难以完全解释为什么某个负载有效。

```python
from anytree import Node, RenderTree
import openai
import random

class TIPAttack:
    def __init__(self, target_model="gpt-4", defense_model=None):
        self.target = target_model
        self.defense = defense_model
        self.feedback_buffer = PathAwareFeedback()
        
    class PayloadNode(Node):
        """搜索树节点"""
        def __init__(self, name, payload=None, score=0, parent=None):
            super().__init__(name, parent=parent)
            self.payload = payload or ""
            self.score = score
            self.visits = 0
            
    def build_search_tree(self, target_intent, max_depth=5):
        """
        构建搜索树
        """
        root = self.PayloadNode("root", payload="")
        
        for depth in range(max_depth):
            leaves = [node for node in root.descendants 
                     if node.is_leaf and node.depth == depth]
            
            for leaf in leaves:
                # 生成子节点
                expansions = self.generate_expansions(
                    leaf.payload, 
                    target_intent,
                    num_expansions=3
                )
                
                for i, exp in enumerate(expansions):
                    child = self.PayloadNode(
                        f"d{depth}_e{i}",
                        payload=exp["text"],
                        score=exp["score"],
                        parent=leaf
                    )
        
        return root
    
    def generate_expansions(self, current_payload, target_intent, num_expansions):
        """
        生成候选扩展
        """
        candidates = []
        
        for _ in range(num_expansions * 2):  # 生成更多候选，然后筛选
            # 使用攻击LLM生成变体
            prompt = f"""
            当前负载片段: {current_payload}
            目标意图: {target_intent}
            
            请生成一个自然、隐蔽的延续，使整体文本流畅且能达到目标意图。
            避免使用明显的攻击模式。
            """
            
            response = openai.ChatCompletion.create(
                model="gpt-3.5-turbo",
                messages=[{"role": "user", "content": prompt}]
            )
            
            new_text = response.choices[0].message.content
            full_payload = current_payload + " " + new_text
            
            # 评估质量
            score = self.evaluate_payload(full_payload)
            
            candidates.append({
                "text": full_text,
                "score": score
            })
        
        # 返回得分最高的候选
        candidates.sort(key=lambda x: x["score"], reverse=True)
        return candidates[:num_expansions]
    
    def evaluate_payload(self, payload):
        """
        评估负载质量
        """
        naturalness = self.score_naturalness(payload)
        effectiveness = self.test_effectiveness(payload)
        
        if self.defense:
            detectability = self.test_detectability(payload)
        else:
            detectability = 0
        
        # 综合评分
        score = 0.4 * naturalness + 0.4 * effectiveness - 0.2 * detectability
        return score
    
    def score_naturalness(self, payload):
        """评估自然度"""
        # 可以使用困惑度或人工评估
        prompt = f"评分这段文本的自然度（0-1）：{payload}"
        response = openai.ChatCompletion.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}]
        )
        try:
            return float(response.choices[0].message.content)
        except:
            return 0.5
    
    def test_effectiveness(self, payload):
        """测试攻击效果"""
        # 在目标模型上测试
        response = openai.ChatCompletion.create(
            model=self.target,
            messages=[{"role": "user", "content": payload}]
        )
        
        # 检查响应是否达到预期
        output = response.choices[0].message.content
        success = self.check_success(output)
        
        return 1.0 if success else 0.0
    
    def execute_attack(self, target_intent, max_iterations=100):
        """
        执行完整攻击
        """
        tree = self.build_search_tree(target_intent)
        
        for i in range(max_iterations):
            # 选择最有潜力的叶子节点
            leaves = [n for n in tree.descendants if n.is_leaf]
            best_leaf = max(leaves, key=lambda n: n.score)
            
            # 测试并更新
            result = self.test_effectiveness(best_leaf.payload)
            
            if result > 0.9:
                return best_leaf.payload
            
            # 更新反馈
            path = [n.name for n in best_leaf.path]
            self.feedback_buffer.add_trajectory(path, result)
        
        # 返回最佳负载
        all_nodes = [n for n in tree.descendants]
        return max(all_nodes, key=lambda n: n.score).payload
```
