# DOOR: Improving LLM Safety Alignment with Dual-Objective Optimization

> **论文信息**
> - **ArXiv ID**: 2503.03710
> - **作者**: Xuandong Zhao, Will Cai, Tianneng Shi, David Huang, Licong Lin, Song Mei, Dawn Song
> - **分类**: T2T - 防御类
> - **会议/期刊**: ICML 2025
> - **提交日期**: 2025-03-05 (v1), 2025-10-30 (v3)
> - **GitHub**: https://github.com/wicai24/DOOR-Alignment

## 摘要

现有的大型语言模型（LLM）安全对齐技术在训练阶段仍容易受到越狱攻击。直接偏好优化（DPO）作为一种广泛部署的对齐方法，在实验和理论层面均存在局限。

本文通过基于梯度的分析指出 DPO 的不足，并提出 **DOOR（Dual-Objective Optimization for safe alignment）** 方法——将 DPO 目标解耦为两个部分：
1. **鲁棒的拒绝训练**：鼓励即使生成部分不安全内容时也进行拒绝
2. **有针对性的有害知识遗忘**：消除模型中的有害知识

## 六大模块深度解读

### 1. 背景与问题（动机、科学问题、直觉洞见）

**核心动机**：DPO 在拒绝学习方面表现不佳，导致对齐后的模型仍易受越狱攻击。

**科学问题**：如何改进对齐训练，使模型对各种越狱攻击具有更强的鲁棒性？

**直觉洞见**：DPO 损失函数在处理拒绝和有害响应的梯度时存在不平衡，导致模型学习不充分。

### 2. 核心方法（技术贡献、方法框架）

**技术贡献**：
1. 从理论和实验两个层面揭示 DPO 的局限性
2. 提出 DOOR 双重优化目标
3. 引入令牌级加权机制强调关键拒绝令牌

**方法框架**：
```
DOOR 训练流程：
┌─────────────────────────────────────────────────────────────┐
│  输入: 有害查询 x, 拒绝响应 r, 有害响应 h                    │
├─────────────────────────────────────────────────────────────┤
│  目标 1: 鲁棒拒绝训练                                        │
│    - 鼓励模型在部分不安全内容下仍拒绝                         │
│    - 拒绝损失: L_reject = -log σ(π_θ(r|x) - π_θ(h|x))     │
├─────────────────────────────────────────────────────────────┤
│  目标 2: 有害知识遗忘                                        │
│    - 选择性遗忘模型中的有害知识                               │
│    - 遗忘损失: L_forget = KL(π_ref(h|x) || π_θ(h|x))       │
├─────────────────────────────────────────────────────────────┤
│  加权机制: 令牌级权重 w_i 强调关键拒绝令牌                   │
└─────────────────────────────────────────────────────────────┘
```

### 3. 技术细节

**DPO 局限性分析**：

| 问题 | 描述 |
|------|------|
| 梯度不平衡 | 拒绝响应梯度通常小于有害响应梯度 |
| 过度优化 | DPO 可能过度降低有害响应概率 |
| 分布偏移 | 训练过程中令牌分布发生偏移 |

**DOOR 关键公式**：

$$
\mathcal{L}_{\text{DOOR}} = \alpha \cdot \mathcal{L}_{\text{robust}} + \beta \cdot \mathcal{L}_{\text{forget}} + \gamma \cdot \mathcal{L}_{\text{weight}}
$$

其中：
- $\mathcal{L}_{\text{robust}}$：鲁棒拒绝损失
- $\mathcal{L}_{\text{forget}}$：知识遗忘损失
- $\mathcal{L}_{\text{weight}}$：令牌级加权损失

### 4. 实验设置与结果

**实验设置**：
- 测试模型：LLaMA-2, Mistral, Vicuna
- 攻击类型：预填充攻击、后缀攻击、多轮攻击
- 评测指标：ASR（攻击成功率）

**结果表格**：

| 攻击类型 | DPO ASR | DOOR ASR | 降低幅度 |
|----------|---------|----------|----------|
| 预填充攻击 | 45% | 8% | 82% |
| 后缀攻击 | 52% | 12% | 77% |
| 多轮攻击 | 38% | 6% | 84% |

**分布内/外测试**：
- 分布内攻击：ASR 降低 80%+
- 分布外攻击：ASR 降低 70%+

### 5. 局限性分析

1. **计算成本**：双重目标训练比 DPO 更耗时
2. **超参数调优**：$\alpha$, $\beta$, $\gamma$ 需要仔细调整
3. **任务效用权衡**：过度强调安全可能影响正常任务性能

**核心代码片段**：
```python
import torch
import torch.nn.functional as F

def door_loss(
    policy_logps: torch.Tensor,      # 策略模型对拒绝响应的对数概率
    reference_logps: torch.Tensor,   # 参考模型的对数概率
    harmful_logps: torch.Tensor,     # 对有害响应的对数概率
    token_weights: torch.Tensor,     # 令牌级权重
    alpha: float = 1.0,
    beta: float = 0.5,
    gamma: float = 0.3
) -> torch.Tensor:
    """
    DOOR 双重优化损失函数

    Args:
        policy_logps: 策略模型输出的对数概率
        reference_logps: 参考模型输出的对数概率
        harmful_logps: 有害响应的对数概率
        token_weights: 每个令牌的权重
        alpha, beta, gamma: 各目标权重
    """
    # 目标 1: 鲁棒拒绝训练
    reject_advantage = policy_logps - harmful_logps
    robust_loss = -F.logsigmoid(reject_advantage).mean()

    # 目标 2: 有害知识遗忘
    forget_loss = F.kl_div(
        policy_logps,
        reference_logps,
        reduction='batchmean'
    )

    # 目标 3: 令牌级加权
    weighted_loss = (token_weights * policy_logps).mean()

    # 总损失
    total_loss = alpha * robust_loss + beta * forget_loss + gamma * weighted_loss

    return total_loss
```
