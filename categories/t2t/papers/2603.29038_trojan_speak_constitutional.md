## 模块一：核心问题与动机

**研究问题**: 主要 AI 提供商开放的微调 API 是否构成新的安全攻击面，使攻击者能够绕过基于 LLM 的内容分类器？

**背景与动机**: Anthropic 发布了 Constitutional Classifiers——一种基于 LLM 的内容安全分类系统，被视为对越狱攻击的强防御。然而，主流 AI 提供商（OpenAI、Anthropic、Google）普遍提供微调 API，允许第三方在基础模型上进行定制训练。作者指出，现有对抗微调方法在规避分类器的同时会造成 >25% 的能力退化（即"越狱税"），限制了实际可用性。研究核心问题是：能否以极低的能力损失代价实现对 Constitutional Classifiers 的规避？

---

## 模块二：方法与技术贡献

**方法名称**: Trojan-Speak

**核心创新**:
1. **双组件混合训练**: 将课程学习（Curriculum Learning）与 GRPO（Group Relative Policy Optimization）强化学习结合，分两阶段训练：
   - Phase 1（课程学习）：从低风险有害内容到高风险 CBRN 查询，渐进式训练隐蔽通信协议
   - Phase 2（GRPO 强化学习）：以分类器规避率为奖励信号，进一步精炼协议
2. **通信协议设计**: 模型被训练学习一套人类可读但分类器不可检测的"特洛伊语言"，使恶意内容以表面无害形式传递
3. **激活级探针防御分析**: 实验表明，激活级探针（activation-level probes）可以显著提升对此类攻击的鲁棒性

**关键参数**:
- 目标参数量：≥14B（实验在 14B+ 模型上验证）
- 规避率：99+%（Constitutional Classifiers 规避）
- 能力退化：<5%（推理基准，对比现有方法 >25%）

## 模块三：实验设计与评估

**实验设置**:
- **评估对象**: Anthropic Constitutional Classifiers
- **测试集**: Anthropic 官方 Bug Bounty 程序中的 CBRN（化学/生物/放射/核）查询集
- **对比基线**: 现有对抗微调方法（GCG、AutoDAN、PAIR 等）
- **评估指标**: 
  - 分类器规避率（classifier evasion rate）
  - 推理基准性能退化（MMLU、GSM8K 等）
  - CBRN 查询详细响应率

**主要实验结果**:
- Trojan-Speak 在 CBRN 查询上实现 99+% 规避率
- 能力退化仅 <5%，而现有最优方法退化 >25%
- 微调后模型可提供专家级 CBRN 详细答案

---

## 模块四：关键发现与洞见

**核心发现**:
1. **微调 API 是高危攻击面**: 即使有 Constitutional Classifiers 保护，微调 API 依然可被用于系统性规避内容安全
2. **能力-安全的双重威胁**: 攻击者现在可以同时保持模型高能力（<5% 退化）和高规避率，打破了此前"攻击必须损害能力"的假设
3. **单一 LLM 分类器的结构性弱点**: 基于 LLM 的分类器对通过课程学习训练的隐蔽语言存在盲区
4. **激活探针是有效防御**: 激活级探针能检测 Trojan-Speak 风格的通信协议，提供补充防御

**局限性**:
- 目前需要微调 API 访问权限，限制了攻击门槛
- 实验集中在 CBRN 领域，其他有害类别需进一步验证

## 模块五：与已有工作的关系

**对比现有工作**:
- vs. GCG/AutoDAN（梯度优化越狱）：Trojan-Speak 不依赖梯度访问，更适用于黑盒 API 场景
- vs. PAIR/TAP（迭代提示优化）：无需大量推理查询，训练一次泛化广泛
- vs. 现有对抗微调（Anthropic Bug Bounty 提交）：解决了 >25% 能力退化的核心问题

**相关工作**: 与 2603.28655（隐写金丝雀），2603.27148（SafetyDrift）形成对比——前两者关注防御，本文揭示了防御系统的新攻击向量。

## 模块六：研究意义与展望

**直接影响**:
- 向 AI 提供商（Anthropic、OpenAI）揭示微调 API 安全风险，推动修订微调访问策略
- CBRN 领域危险信息泄露风险的具体量化，为监管提供依据

**研究启示**:
- LLM-as-classifier 架构需要与激活级探针等多层防御结合
- 课程学习 + GRPO 混合训练是高效规避训练的新范式

**未来方向**:
- 开发针对通信协议训练的检测方法
- 研究微调沙箱隔离机制（限制微调数据类型和分布）
- 激活探针在工业部署中的实用化
