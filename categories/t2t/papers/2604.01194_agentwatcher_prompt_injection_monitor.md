## 模块一：研究背景与问题定义

大语言模型（LLM）及其Agent应用面临严重的**提示注入（Prompt Injection）**攻击威胁。攻击者通过在外部文档、网页或工具返回内容中嵌入恶意指令，操控Agent执行非预期操作。现有检测方法存在两大关键缺陷：其一，随着上下文长度增加，检测性能大幅下降；其二，缺乏对"何为提示注入"的明确规则定义，导致检测决策不透明、难以向用户解释。

**核心问题**: 如何设计一个在长上下文环境中仍可扩展、且具有可解释性的提示注入检测系统？

---

## 模块二：方法论与技术框架

AgentWatcher 提出了两项关键技术创新：

**1. 因果归因机制（Causal Attribution）**  
将LLM的输出（Agent动作）归因于上下文中少数具有因果影响力的片段（Influential Segments）。通过聚焦于这些关键片段而非全部上下文，实现长文本下的可扩展检测。技术上，借助影响函数（Influence Functions）或梯度归因方法，为每个上下文片段计算其对Agent决策的贡献分数 s_i ∈ ℝ，仅保留 Top-K 高影响片段进行后续分析。

**2. 规则驱动检测框架（Rule-based Detection）**  
形式化定义提示注入判定规则，通过监控LLM对这些规则进行推理，生成结构化且可解释的检测决策。规则包含：
- 意图分解（Intent Decomposition）：分析片段的实际意图与系统预期的偏差
- 安全信号验证（Safety Signal Verification）：检查是否含有命令覆盖模式
- 危害评估（Harm Assessment）：量化潜在危害等级
- 自反思（Self-Reflection）：二次验证检测结论

**整体流程**: 原始上下文 → 因果归因过滤 → 规则推理监控 → 检测决策 + 解释

## 模块三：核心实验结果

**实验设置**: 在工具使用Agent基准测试（Tool-Use Agent Benchmarks）和长上下文理解数据集上进行评估，与主流检测方法对比。

**关键结果**:
- 在长上下文场景（上下文长度 > 8K tokens）下，AgentWatcher 的检测精度相比基线方法提升显著，解决了"长上下文衰减"问题
- 在无攻击场景（Clean Setting）下保持了高实用性，误报率低
- 可解释性评估显示，规则驱动决策使安全审计人员能够快速定位攻击来源

**局限性**: 因果归因计算带来一定推理开销；规则定义依赖专家知识，对新型攻击模式的泛化性需进一步验证。

## 模块四：与相关工作对比

| 方法 | 长上下文鲁棒性 | 可解释性 | 计算效率 |
|------|--------------|---------|---------|
| 基于分类器的检测 | 弱（随上下文增长显著下降） | 无 | 高 |
| LLM-as-a-Judge（2603.25176）| 中 | 有 | 中 |
| **AgentWatcher** | **强（通过归因聚焦关键片段）** | **高（规则驱动+结构化解释）** | 中低 |

相比同期论文 2603.25176 (Prompt Attack Detection with LLM-as-a-Judge)，AgentWatcher 在可扩展性上更进一步，专门解决了长上下文衰减问题，而不仅依赖LLM的判断能力。

## 模块五：实践意义与应用价值

**直接应用场景**:
- 企业级LLM Agent部署的实时安全监控
- RAG系统中的检索内容安全过滤
- 多轮对话Agent中的持续性注入检测

**防御价值**: AgentWatcher 的因果归因思路与"预测与追踪"防御趋势（SafetyDrift、Kill-Chain Canaries）一脉相承，均强调在攻击产生效果前进行精准干预。规则驱动框架则为Agent安全治理（2603.26221 Clawed and Dangerous）提供了可操作的实施路径。

**开源影响**: 代码开源有助于推动社区建立统一的提示注入检测标准，类似于 2603.24857 中呼吁的统一威胁分类法框架。

---

## 模块六：局限与未来方向

**当前局限**:
- 因果归因方法的计算成本较高，实时部署存在延迟
- 规则集需要持续更新以覆盖新型攻击变体（如多跳注入、隐蔽编码注入）
- 在高度动态的Agent工作流（如代码执行、浏览器控制）中尚未充分验证

**未来方向**:
1. 自适应规则学习：通过在线学习动态扩展规则库
2. 轻量化归因：探索近似归因方法降低推理开销
3. 多智能体扩展：将AgentWatcher扩展到多Agent系统（参考 2603.26221 的治理框架）

**研究分类**: Defense | Prompt Injection | LLM Agent Security
