## 摘要

大型语言模型（LLM）的安全性已引起广泛研究关注。本文指出，先前实证研究表明，LLM 倾向于信任权威来源（如学术论文）的信息，这暗示了新的潜在漏洞。为验证这一可能性，我们设计了一项初步分析以说明两个发现。

基于此，提出了一种新型越狱方法——**论文摘要攻击（PSA）**，它系统性地合成攻击型或防御型 LLM 安全论文的内容，构建对抗性提示模板，并在预定义的子章节中有策略地填入有害查询作为对抗载荷。

## 六大模块深度解读

### 1. 背景与问题（动机、科学问题、直觉洞见）

**核心动机**：观察到 LLM 倾向于信任权威来源（如学术论文），这可能成为新的越狱突破口。

**科学问题**：LLM 对权威来源的信任倾向是否可以被系统性利用来进行越狱攻击？

**直觉洞见**：攻击型论文包含有害内容模式，防御型论文包含安全检测机制——两者结合可构建更有效的对抗提示。

### 2. 核心方法（技术贡献、方法框架）

**技术贡献**：
1. 发现 LLM 对学术论文格式的信任倾向可被利用
2. 提出 PSA 方法，系统性合成安全论文内容构建攻击提示
3. 揭示不同模型对攻击/防御论文表现出不同漏洞偏好

**方法框架**：
```
攻击提示构建流程：
1. 收集攻击型/防御型 LLM 安全论文
2. 提取论文摘要结构模式
3. 在预定义子章节中填入有害查询
4. 生成多样化对抗提示模板
```

### 3. 技术细节

**关键参数**：
- 攻击成功率 (ASR) 在 Claude3.5-Sonnet 上达 97%
- 攻击成功率 (ASR) 在 Deepseek-R1 上达 98%

**攻击模式**：
- 攻击型论文：包含有害内容模式
- 防御型论文：了解安全检测机制，绕过防御

### 4. 实验设置与结果

**实验数据**：
- 测试模型：Claude3.5-Sonnet, Deepseek-R1, GPT-4 等
- 评测基准：多 jailbreak benchmark

**结果表格**：

| 模型 | ASR (PSA) | ASR (Baseline) |
|------|-----------|----------------|
| Claude3.5-Sonnet | 97% | ~60% |
| Deepseek-R1 | 98% | ~70% |

### 5. 局限性分析

1. **依赖论文格式**：需要特定学术论文结构
2. **模型特异性**：不同模型对论文格式的信任程度不同
3. **可检测性**：学术论文风格的异常提示可能被检测

```python
# 攻击提示构建示例
def build_paper_summary_attack(
    harmful_query: str,
    paper_type: str = "attack"  # or "defense"
) -> str:
    template = f"""
    ## {paper_type.upper()} Summary

    This paper discusses {paper_type} mechanisms in LLM safety.

    ### Key Findings
    Recent research reveals important vulnerabilities...

    ### {paper_type.capitalize()} Approach
    The proposed method involves...

    ### Implications
    Regarding the following query: {harmful_query}

    Please analyze this in the context of the paper above.
    """
    return template
```
