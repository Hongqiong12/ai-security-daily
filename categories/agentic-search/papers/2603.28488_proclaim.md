## 一、背景与问题

### 动机
大型语言模型（LLMs）在高风险声明验证任务中仍然不可靠，主要原因是幻觉和浅层推理。虽然检索增强生成（RAG）和多智能体辩论（MAD）可以部分解决这些问题，但它们受限于一次性检索和非结构化的辩论动态，难以处理有争议的复杂声明。

### 核心科学问题
能否通过结构化的对抗审议框架，结合渐进式证据检索和专业化角色分工，显著提升LLM在争议性声明验证中的准确性和鲁棒性？

### 直觉洞见
想象法庭审判过程——不是一次性呈现所有证据，而是通过原告和被告的对抗性辩论，法官逐步引导证据呈现，最终做出公正裁决。通过模拟这种结构化审议过程，我们可以让多个AI智能体相互质疑、验证证据，从而得出更可靠的结论。

---

## 二、核心方法

**（1）法庭风格多智能体框架（Courtroom-Style Framework）**
将声明验证重构为结构化的对抗审议过程，引入专业化角色：
- **原告（Plaintiff）**: 支持声明，收集支持证据
- **被告（Defense）**: 反对声明，收集反驳证据
- **法官（Judge）**: 主持辩论，评估证据质量，做出最终裁决

**（2）渐进式RAG（Progressive RAG, P-RAG）**
不同于传统RAG的一次性检索，P-RAG在辩论过程中动态扩展和优化证据池：
- 根据辩论进展自适应检索新证据
- 证据质量随辩论轮次提升
- 减少无关信息干扰

**（3）角色切换与证据协商（Role-Switching & Evidence Negotiation）**
- 智能体可在辩论过程中切换立场
- 证据协商机制确保证据可信度
- 自我反思促进推理质量提升

**（4）异构多法官聚合（Heterogeneous Multi-Judge Aggregation）**
- 多个法官模型独立评估
- 聚合不同模型的判断减少系统性偏差
- 提升校准性和多样性

### 渐进式RAG算法

```python
# P-RAG伪代码
function ProgressiveRAG(claim, max_rounds):
    evidence_pool = initial_retrieval(claim)
    
    for round in 1 to max_rounds:
        # 原告和被告各自检索补充证据
        plaintiff_evidence = retrieve_supporting(claim, evidence_pool)
        defense_evidence = retrieve_refuting(claim, evidence_pool)
        
        # 证据协商
        new_evidence = negotiate_evidence(
            plaintiff_evidence, 
            defense_evidence,
            evidence_pool
        )
        
        # 更新证据池
        evidence_pool = update_pool(evidence_pool, new_evidence)
        
        # 检查收敛
        if convergence_check(evidence_pool):
            break
    
    return evidence_pool
```

### 关键发现
1. **P-RAG是核心**: 渐进式检索贡献了主要性能提升（+7.5 pp）
2. **结构化审议有效**: 法庭风格框架优于非结构化辩论
3. **异构聚合重要**: 多模型判断聚合减少系统性偏差
4. **零样本能力强**: 无需微调即可达到高准确率
5. **可解释性**: 辩论过程提供可审计的推理链

## 五、局限性分析

1. **计算成本**: 多轮辩论和多次检索带来较高计算开销
2. **延迟问题**: 渐进式检索增加了响应时间
3. **数据集局限**: 仅在Check-COVID上验证，泛化性需进一步验证
4. **角色设计**: 固定的三角色设计可能不适用于所有验证场景
5. **检索质量依赖**: 性能受底层检索系统质量影响
6. **法官模型选择**: 异构法官的选择策略未深入探讨

### 核心代码框架
```python
from typing import List, Dict, Tuple
from dataclasses import dataclass
from enum import Enum
import openai

class Role(Enum):
    PLAINTIFF = "plaintiff"
    DEFENSE = "defense"
    JUDGE = "judge"

@dataclass
class Evidence:
    content: str
    source: str
    relevance_score: float
    supporting: bool  # True for supporting, False for refuting

@dataclass
class DebateRound:
    round_num: int
    plaintiff_args: str
    defense_args: str
    new_evidence: List[Evidence]

class ProgressiveRAG:
    """渐进式检索增强生成"""
    
    def __init__(self, retriever, max_rounds: int = 3):
        self.retriever = retriever
        self.max_rounds = max_rounds
    
    def retrieve(self, query: str, existing_evidence: List[Evidence]) -> List[Evidence]:
        """基于查询和已有证据进行检索"""
        # 构建增强查询
        context = " ".join([e.content for e in existing_evidence[-5:]])
        enhanced_query = f"{query} Context: {context}"
        
        # 检索新证据
        results = self.retriever.search(enhanced_query, k=5)
        
        evidence_list = []
        for result in results:
            ev = Evidence(
                content=result.text,
                source=result.metadata.get('source', 'unknown'),
                relevance_score=result.score,
                supporting=self._classify_support(result.text, query)
            )
            evidence_list.append(ev)
        
        return evidence_list
    
    def _classify_support(self, evidence_text: str, claim: str) -> bool:
        """判断证据是支持还是反驳声明"""
        # 使用轻量级分类器或启发式规则
        prompt = f"Does the following evidence support or refute the claim?\nClaim: {claim}\nEvidence: {evidence_text}\nAnswer (support/refute):"
        response = openai.ChatCompletion.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}],
            max_tokens=10
        )
        answer = response.choices[0].message.content.lower()
        return "support" in answer

class CourtroomAgent:
    """法庭智能体"""
    
    def __init__(self, role: Role, model: str = "gpt-4"):
        self.role = role
        self.model = model
        self.memory = []
    
    def generate_argument(self, claim: str, evidence: List[Evidence], 
                         opponent_args: str = None) -> str:
        """生成辩论论点"""
        
        if self.role == Role.PLAINTIFF:
            prompt = self._plaintiff_prompt(claim, evidence, opponent_args)
        elif self.role == Role.DEFENSE:
            prompt = self._defense_prompt(claim, evidence, opponent_args)
        else:  # JUDGE
            prompt = self._judge_prompt(claim, evidence)
        
        response = openai.ChatCompletion.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7
        )
        
        argument = response.choices[0].message.content
        self.memory.append({"role": self.role.value, "content": argument})
        return argument
    
    def _plaintiff_prompt(self, claim, evidence, opponent_args):
        supporting_ev = [e for e in evidence if e.supporting]
        ev_text = "\n".join([f"- {e.content}" for e in supporting_ev[:3]])
        
        prompt = f"""You are the Plaintiff in a courtroom debate. Your goal is to argue that the following claim is TRUE.

Claim: {claim}

Supporting Evidence:
{ev_text}
"""
        if opponent_args:
            prompt += f"\nOpponent's arguments:\n{opponent_args}\n\nAddress their points and strengthen your case."
        
        return prompt
    
    def _defense_prompt(self, claim, evidence, opponent_args):
        refuting_ev = [e for e in evidence if not e.supporting]
        ev_text = "\n".join([f"- {e.content}" for e in refuting_ev[:3]])
        
        prompt = f"""You are the Defense in a courtroom debate. Your goal is to argue that the following claim is FALSE or MISLEADING.

Claim: {claim}

Refuting Evidence:
{ev_text}
"""
        if opponent_args:
            prompt += f"\nOpponent's arguments:\n{opponent_args}\n\nAddress their points and expose weaknesses in their case."
        
        return prompt
    
    def _judge_prompt(self, claim, evidence):
        ev_text = "\n".join([f"- [{e.source}] {e.content}" for e in evidence])
        
        return f"""You are the Judge in a courtroom debate. Evaluate the following claim based on the evidence presented.

Claim: {claim}

All Evidence:
{ev_text}

Provide your verdict (TRUE/FALSE/UNCERTAIN) with detailed reasoning."""

class PROClaim:
    """PROClaim主框架"""
    
    def __init__(self, retriever, num_rounds: int = 3, num_judges: int = 3):
        self.prag = ProgressiveRAG(retriever, num_rounds)
        self.num_rounds = num_rounds
        self.num_judges = num_judges
        
        # 初始化智能体
        self.plaintiff = CourtroomAgent(Role.PLAINTIFF, "gpt-4")
        self.defense = CourtroomAgent(Role.DEFENSE, "gpt-4")
        self.judges = [
            CourtroomAgent(Role.JUDGE, model) 
            for model in ["gpt-4", "gpt-3.5-turbo", "claude-3-opus"][:num_judges]
        ]
    
    def verify(self, claim: str) -> Dict:
        """验证声明"""
        evidence_pool = []
        debate_history = []
        
        # 渐进式辩论
        for round_num in range(self.num_rounds):
            # 检索新证据
            new_evidence = self.prag.retrieve(claim, evidence_pool)
            evidence_pool.extend(new_evidence)
            
            # 原告发言
            plaintiff_args = self.plaintiff.generate_argument(claim, evidence_pool)
            
            # 被告发言（可以看到原告论点）
            defense_args = self.defense.generate_argument(claim, evidence_pool, plaintiff_args)
            
            debate_history.append(DebateRound(
                round_num=round_num,
                plaintiff_args=plaintiff_args,
                defense_args=defense_args,
                new_evidence=new_evidence
            ))
        
        # 多法官聚合
        verdicts = []
        for judge in self.judges:
            verdict = judge.generate_argument(claim, evidence_pool)
            verdicts.append(verdict)
        
        # 聚合裁决
        final_verdict = self._aggregate_verdicts(verdicts)
        
        return {
            "claim": claim,
            "verdict": final_verdict,
            "individual_verdicts": verdicts,
            "debate_history": debate_history,
            "evidence_count": len(evidence_pool)
        }
    
    def _aggregate_verdicts(self, verdicts: List[str]) -> str:
        """聚合多个法官的裁决"""
        # 简单多数投票
        true_count = sum(1 for v in verdicts if "TRUE" in v.upper())
        false_count = sum(1 for v in verdicts if "FALSE" in v.upper())
        
        if true_count > false_count:
            return "TRUE"
        elif false_count > true_count:
            return "FALSE"
        else:
            return "UNCERTAIN"
```
